# AI Debt Scanner Skill 🚀

A skill designed to audit and clean technical debt generated by AI ("vibe coding") in multi-language projects. It uses structural analysis heuristics to detect common patterns in LLM-generated code, such as dead code blocks, monolithic functions, AI meta-talk, and poor exception handling.

## Features

- 🔍 **Language Agnostic:** Supports Python, TypeScript, JavaScript, Go, Rust, Java, and C++.
- ⚖️ **Debt Scoring:** Evaluates each file and the overall project using a weighted scoring system.
- 🤖 **AI Artifact Detection:** Finds traces of AI conversations, excuses, or leftovers ("as an AI model...", "I can help with...").
- 📦 **TOON Support (Token-Oriented Object Notation):** Output optimized to reduce token consumption in LLMs by ~40%.
- 🤝 **Multi-Agent Architecture:** Modular system with specialized agents (Scanner, Architect, Cleaner).

## Project Structure

```
.
├── SKILL.md                 # Skill definition and multi-agent protocol
├── agents/                  # Specialized agents (Scanner, Architect, Cleaner)
├── scripts/
│   └── scan.py              # Main scanning script with TOON/JSON output
├── references/              # Rules and refactoring patterns
└── tests/                   # Sample code with debt for validation
```

## Usage

To scan a directory, run the scanning script with Python:

```bash
python3 scripts/scan.py /path/to/your/project
```

If no path is specified, it scans the current directory by default:

```bash
python3 scripts/scan.py .
```
